Welcome, Dreamworld!

This is the Django/Python Web App made by Glen Baker for the Dreamworld technical test

This site is using the built-in Django user management as outlined in the test - so you can login at '/admin' or by clicking on Login above. You can peruse the city data from there, but you'll want to come back to list of cities.

  • To load the data from JSON I wrote the script, cities/load_data_from_sba.py
  • city_list has a basic search bar for filtering through the names of the cities
  • city_detail template displays all of the city's information as well as a Google Map centered on the coordinates.
